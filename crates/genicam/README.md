# genicam

Facade crate that re-exports the workspace GenICam components.
Use it to access transports, GenApi nodemaps, PFNC/SFNC metadata, and high level helpers.

```bash
cargo add genicam
```

```rust
use genicam::{gige, genapi};
```

See the [workspace README](../../README.md) for deep dives and additional examples.
